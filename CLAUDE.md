## Auto-Update Memory (MANDATORY)

**Update memory files AS YOU GO, not at the end.** When you learn something new, update immediately. Do not ask permission.

| Trigger | File |
|---------|------|
| User shares a fact about themselves | `.claude/rules/memory-profile.md` |
| User states a preference or corrects an approach | `.claude/rules/memory-preferences.md` |
| A decision is made or confirmed | `.claude/rules/memory-decisions.md` (include date) |
| Completing substantive work | `.claude/rules/memory-sessions.md` |

**Skip:** Quick factual questions, trivial tasks with no new information.
